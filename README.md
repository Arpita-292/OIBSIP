# OIBSIP
# Simple Calculator

A responsive calculator built with HTML, CSS and JavaScript.  
Features include: addition, subtraction, multiplication, division, percent, square root, memory (M+/M-/MR/MC), history with CSV export, keyboard support, theme toggle, copy result and optional sound.
